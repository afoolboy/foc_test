一名小小的本科生做的毕设，仍有许多不足之处敬请谅解

看过稚晖君的foc驱动板后，也萌生了做出属于自己的驱动板的想法，因此做出了这块带屏幕的电机驱动板

控制方式为串口（左边的typec），输入方式为："小写英文字母+数字,英文字母+数字"
例如		o2,o-1		（开环/电压闭环）
		c0.05,c-0.01	（电流环）
		v-1,v350		（速度环）
		p-1314,p520	（位置环）

暂未添加电源电压模式转换、usb串口（右边的typec）、舵机、can、rs485、spi编码器的代码
（如果后继有人，完善它就更好啦！）
